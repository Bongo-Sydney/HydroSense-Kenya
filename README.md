# HydroSense-Kenya

## Scientific Computing System for Smart Irrigation

**Course:** ICS 2207 Scientific Computing  
**Semester:** February-May 2026

---

## Project Overview

HydroSense-Kenya is a scientific computing system that estimates soil-water balance, detects water stress, simulates uncertainty, and recommends irrigation decisions for Kenyan smallholder farms in form of irrigation schedules.

### Key Features
- Water balance simulation using Euler and Runge-Kutta methods
- Root finding for irrigation amount calculation (Bisection, Newton-Raphson, Secant)
- Numerical integration for cumulative water deficit (Trapezoidal, Simpson's 1/3)
- Monte Carlo simulation for rainfall uncertainty
- Optimized irrigation scheduling under water constraints
